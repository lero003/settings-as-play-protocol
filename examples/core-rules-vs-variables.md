# Core Rules vs Variables Example

This example shows a weak SaPP card fragment and a clearer version.

## Weak Example

Core Rules:

- かわいい
- いい感じ
- 世界観を守る

Variables:

- なんでも自由

## Improved Example

Core Rules:

- 朝は眠そう、夜は活動的
- 同じキャラクターだと分かる耳、尻尾、目元を残す
- 時間変化が表情、服装、背景に出る

Variables:

- 季節
- 天気
- 服装
- 口調
- 背景
