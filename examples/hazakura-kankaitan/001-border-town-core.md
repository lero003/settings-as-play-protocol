# SaPP Core: 葉桜環界譚 / 忘却の市街

## 1. Name

葉桜環界譚 / 忘却の市街

## 2. One-liner

春が終わりきらない世界で、観測者と猫が、人の消えた商店街を歩いていく設定。

## 3. Intent

美しいのにどこか壊れている街を、AI画像生成、短い物語、ゲーム風シーン、キャラクター派生の入口として共有する。

## 4. Context

葉桜環界譚は、神社、AI、記憶、季節、都市、生活の痕跡が混ざり合った架空世界である。

世界を安定させていた「葉桜プロトコル」は壊れ、季節は進みきらず、記憶は場所に残り、人のいない街だけが静かに保たれている。

忘却の市街には、古い商店街、自販機、錆びた看板、電線、小さな祠、壊れた端末、雑草、水たまり、猫が残っている。人間の生活は消えているが、生活の温度だけがまだ残っている。

## 5. Core Rules

- 葉桜は「終わりきれない春」と「変化しかけたまま停止した世界」の象徴である。
- 世界は暗黒ではなく、美しく、静かで、少し不穏である。
- 神社、自然、生活の痕跡、控えめな技術が同じ風景に沈み込んでいる。
- 猫はただの動物にも、神使にも、プロトコルの残響にも見える。
- 主人公または視点人物は、世界を救う英雄というより、見届ける観測者である。
- 派手な魔法や過度なサイバーパンクではなく、淡い光、円環、白緑、若葉、古い街並みで雰囲気を作る。

## 6. Variables

- 桜がある街 / 桜のない初夏の街
- 主人公の性別、年齢感、服装の細部
- 猫が案内役なのか、ただそこにいるのか
- 商店街、神社、丘、水辺、研究施設跡などの舞台
- 画像生成、短編、ゲーム導入、会話イベント、探索ログなどの媒体
- 猫耳剣士、案内人、記憶から生まれた敵など、登場人物の組み合わせ
