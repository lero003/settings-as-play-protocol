# Minimal SaPP Core: 雨の日だけ歌う灯台

This is a tiny SaPP Core draft. It only fills the fields needed to start.

## 1. Name

雨の日だけ歌う灯台

## 2. One-liner

雨の日だけ、海に向かって古い歌を流す灯台の設定。

## 5. Core Rules

- 灯台は雨の日だけ歌う
- 歌は誰かを呼ぶためではなく、何かを忘れないために流れる
- 海辺の町の人々は、その歌を少し怖がりながら受け入れている

## 6. Variables

- 歌の由来
- 灯台がある町
- 歌を聞く人物
- 雨の強さ
- 歌が聞こえる範囲
- 画像、短編、ゲーム導入、音声作品などの媒体
