# Character-Focused Sample: 朝夜で変わる猫耳ちゃん

This sample shows how a simple SaPP Core can grow into a character-focused card.

## SaPP Core Summary

### Name

朝夜で変わる猫耳ちゃん

### One-liner

朝は寝ぼけていて、夜は活動的になる猫耳キャラクター設定。

### Core Rules

- 同じキャラクターであることがわかる
- 朝は眠そう、夜は活動的
- 耳や尻尾に時間帯の雰囲気が出る
- 変化はかわいさや物語性につながる

### Variables

- 服装
- 背景
- 季節
- 天気
- 口調
- 曜日や月齢との組み合わせ

## Character Module

### Appearance

猫耳と尻尾を持つキャラクター。朝は柔らかい部屋着や寝ぐせ、夜は軽い外出着やきらっとした目元で変化を出す。

### Personality

朝は反応が遅く、少しぼんやりしている。夜は好奇心が強く、少しだけいたずらっぽい。

### Relationship

見る人や一緒にいる相手によって、朝は甘え気味、夜は案内役のように振る舞える。

### Change Pattern

時間帯で表情、姿勢、服装、背景、耳と尻尾の動きが変わる。

### Do / Do Not

Do:

- 同じキャラクターだとわかる要素を残す
- 朝と夜の差を表情や姿勢で見せる
- 時間帯の変化を楽しく扱う

Do Not:

- 朝と夜で別人に見えるほど変えない
- 猫耳だけを記号として扱い、時間変化の面白さを消さない
- 過度に性的な表現へ寄せない
