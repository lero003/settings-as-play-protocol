# SaPP Core: 朝夜で変わる猫耳ちゃん

## 1. Name

朝夜で変わる猫耳ちゃん

## 2. One-liner

朝は寝ぼけていて、夜は活動的になる猫耳キャラクター設定。

## 3. Intent

同じキャラクターが時間帯によって変化する楽しさを、画像生成、短い場面、派生設定で遊ぶ。

## 4. Context

日常の中にいる猫耳キャラクター。朝・昼・夜の変化が、表情、服装、背景、行動に現れる。

## 5. Core Rules

- 同じキャラクターであることがわかる
- 朝は眠そう、夜は活動的
- 耳や尻尾に時間帯の雰囲気が出る
- 変化はかわいさや物語性につながる

## 6. Variables

- 服装
- 背景
- 季節
- 天気
- 年齢感
- 口調
- 朝夜以外の時間帯
- 曜日、月齢、感情との組み合わせ
