# Derivation Map

Derivation Map is a draft idea for showing how settings branch from one another.

It is not required to use SaPP.

## Minimal Example

```text
Original Setting
├─ Derivative A: changed season and setting
├─ Derivative B: changed medium from image prompt to short story
└─ Derivative C: kept core rules, changed character role
```

## What To Track

- Source setting
- Derivative setting
- What changed
- What stayed
- Credit link

This document is a placeholder for future examples. For now, use [Derivation Card](../templates/derivation-card.md) when recording a derivative.
