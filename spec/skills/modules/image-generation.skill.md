# Image Generation Module Skill

## Purpose

Use this module when a SaPP setting will guide AI image generation, visual exploration, reference sheets, posters, concept art, or visual continuity.

The Image Generation Module should make visual reuse easier while preserving creative and rights boundaries.

## Sections

### Visual Keywords

Reusable visual keywords, motifs, style cues, colors, objects, character features, and setting signals.

### Composition

Camera, framing, distance, pose, layout, foreground, background, movement, and visual hierarchy.

### Mood

Emotional tone, light, season, time of day, texture, atmosphere, and visual energy.

### Prompt Notes

Prompt hints, ordering, model-specific notes, reusable phrases, and prompt fragments.

Prompt Notes are advisory and may need adjustment for each image model.

### Negative Constraints

Elements to avoid because they break setting identity, tone, rights boundaries, safety boundaries, or visual continuity.

### Example Prompts

Optional short examples that demonstrate reusable generation direction.

Example prompts should not be treated as the only valid expression of the setting.

### Asset Notes

Use when publishing generated or reference assets.

Expected fields:

- Purpose
- Generated by
- Legal License / Permission
- Canonical Status
- Boundary

## Validation Notes

Warn if Visual Keywords or Prompt Notes contradict Core Rules.

Warn if Negative Constraints contain legal or permission claims that should also appear in Derivation & Credit.

Warn if AI-generated assets do not declare Canonical Status.

Warn if model-specific prompt language is presented as stable SaPP canon.
