# Worldbuilding Module Skill

## Purpose

Use this module when a SaPP setting needs place, society, history, atmosphere, or shared world rules.

The Worldbuilding Module should help an AI assistant generate consistent scenes and adaptations without inventing hidden canon.

## Sections

### Place

Location, geography, environment, scale, landmarks, routes, and spatial relationships.

### Era

Time period, civilization level, technology, calendar, daily rhythm, and historical position.

### Social Rules

Customs, law, taboo, institutions, power structures, class, etiquette, and everyday common sense.

### Mystery

Hidden premises, unresolved questions, contradictions, secrets, myths, or unknown mechanisms.

Mystery should be marked as unknown when the answer is intentionally unresolved.

### Symbols

Motifs, colors, objects, weather, plants, architecture, sounds, rituals, and repeated sensory cues.

## Validation Notes

Warn if Social Rules conflict with SaPP Core Context or Core Rules.

Warn if Mystery is resolved by the assistant without user instruction.

Warn if Symbols are treated as mandatory when they are only mood references.
